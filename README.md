# Global Weather Analytics Dashboard

An interactive data visualization dashboard analyzing 140,000+ real weather records across 200+ countries.

## Features
- Live world heatmap with temperature, humidity, wind speed and air quality visualization
- Temperature distribution analysis
- Humidity vs Temperature scatter plot
- Top 10 hottest countries comparison
- Wind speed distribution and UV index analysis
- Country-wise CO levels
- Interactive filters by country and weather parameter

## Tech Stack
Python, Pandas, Plotly, Streamlit
